# premierleague-flanksource

## Installation

Clone the repo:

```bash
git clone git@github.com:geraldtivatyi/premierleague-flanksource.git
```

Install all project dependencies

```bash
npm install / npm install --force
```

## Running Locally

Run project

```bash
npm start
```

## Running Unit Tests

Run all unit tests

```bash
npm test
```
